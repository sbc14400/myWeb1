Event

  • this 를 이해해라..
  • event target : button, text 등의 control (event가 발생하는 대상)
  • event type : onclick, onchange
  • event handler : event가 발생할때 실행되는 code, function
  • inline 방식
    1. : tag안에 event code를 작성
    2. onclick="alert()"
    3. 장점 : 간단
    4. 단점 : html은 css(설계), js(제어)로 분리하여 정보를 표현하는 것이 방향
  • property listener 방식
    1. script에 element 객체를 찾아서 event handler 작성
    2. t.onclick = function(event){ }

  • addEventListener 방식

    1. 가장 권장, 복수의 Element에 하나의 리스트를 등록해서 재사용가능
    2. t2.addEventListener('click', function(event){ });
    3. t2.addEventListener('click', fun_name); + switch case
